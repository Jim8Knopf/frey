To access Frey services securely, you need to install the Frey Root Certificate Authority on your device.

Download Certificate

Download the certificate directly from your Frey host:

Download Frey Root CA

Why install the CA?

  • Enables trusted HTTPS for all Frey services
  • Allows mobile apps to sync without certificate prompts
  • Powers secure authentication with Authentik

Android Installation

  1. Download the certificate file
  2. Open Settings → Security & privacy → More security settings → Encryption & credentials
  3. Tap Install certificate → CA certificate
  4. Select the downloaded Frey certificate file
  5. Reboot your device

Verify: Settings → Trusted credentials (User tab) should show "Frey Root CA"

Linux Installation

  1. Copy the certificate to /usr/local/share/ca-certificates/frey-root-ca.crt
  2. Run sudo update-ca-trust
  3. Restart your browser

Verify: Run trust list | grep "Frey Root"

Windows Installation

  1. Double-click the downloaded certificate file
  2. Click "Install Certificate"
  3. Select "Local Machine" and click Next
  4. Choose "Place all certificates in the following store"
  5. Browse and select "Trusted Root Certification Authorities"
  6. Click Next and Finish

macOS/iOS Installation

  1. Download and open the certificate file
  2. macOS: Add to Keychain, then open Keychain Access → Certificates → Right-click certificate → Get Info → Trust → Always Trust
  3. iOS: Settings → Profile Downloaded → Install → Enter passcode → Install
  4. iOS: Settings → General → About → Certificate Trust Settings → Enable for Frey Root CA
